# Wammy
This is a simple project with most of the jetpack packages and retrofit 2.6.0, it was built for learning purpose and will be updated time to time

## Built with
* Shared Element
* Retrofit 2.6.0
* Coroutine
* Live Data
* View Model
* Repository Pattern
* MVVM architecture
* Androidx
* Viewbinding
* Material design components
* Kotlin
* Shimmer effect
* Leak Canary2
* Kotlin Flow
