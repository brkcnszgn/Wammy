# Wammy
This is a simple project with most of the jetpack packages and retrofit 2.6.0, it was built for learning and will also be updated

## Built with
* Shared Element
* Retrofit 2.6.0
* Coroutine
* Live Data
* View Model
* Repository Pattern
* MVVM architecture
* Androidx
* viewbinding
* Material design components
* kotlin
* shimmer effect
